First Detector Bot

Telegram-бот для анализа сетевого трафика из файлов .pcap / .pcapng, полученных с помощью Wireshark.
Бот автоматически анализирует пакеты и выявляет возможные признаки сетевых атак.

Возможности
1) Приём файлов .pcap / .pcapng
2) Анализ сетевого трафика
3) Обнаружение подозрительной активности
4) Простая система подписки
5) Поддержка двух языков (RU / EN)
6) Удобный интерфейс через Telegram

Как это работает
1. Пользователь запускает бота (/start)
2. Выбирает язык
3. Оформляет подписку
4. Загружает файл сетевого трафика
5. Бот анализирует пакеты и отправляет отчёт

Поддерживаемые типы анализа

Сейчас бот проверяет несколько базовых признаков атак:
SYN Flood (DoS): анализ большого количества SYN-пакетов
ARP Spoofing: анализ количества ARP пакетов
